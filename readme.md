# People Counting with YOLO and OpenCV

This project implements people counting using YOLO (You Only Look Once) object detection with OpenCV. The script tracks people entering and exiting predefined areas.

## Requirements

- Python 3.x
- OpenCV
- Pandas
- NumPy
- Ultralytics
- Trackertt
